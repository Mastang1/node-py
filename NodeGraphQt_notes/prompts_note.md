## 1. 让AI自己闭环的prompt
 - short：
 ```
 写完后你必须：在本机运行该程序，根据报错或异常自行修复，直到能正常运行并验证功能正常，再交付。不要只交代码不运行。
 ```

 - full：
 ```
 开发完成后必须完成验证闭环再交付：
1. 在本地/工作区执行我让你写的程序（或相关入口命令）；
2. 若运行报错、崩溃或行为不符合需求，自行根据报错和需求修改代码并重新运行；
3. 重复步骤 2，直到程序能正常启动且核心功能可用；
4. 交付时说明：已运行验证，并简要列出验证方式（例如“已执行 python xxx，窗口正常、RUN 可点击”）。
禁止：只提交代码而不运行、不修复运行错误就结束。
```
